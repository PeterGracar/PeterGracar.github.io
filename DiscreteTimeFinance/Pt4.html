Definition 1.2.22.

A financial market model is called complete, if for any contingent claim XX there exists a replicating strategy (x,ϕ)(x,\phi). A model which is not complete is called incomplete.

By Proposition 1.2.17 the issue of computing prices in complete market models is completely solved. But how to recognise complete models? The following proposition gives us a criterion for completeness.

Proposition 1.2.23.

Assume a general single period market model consisting of stocks S1,…,SnS^{1},...,S^{n} and a money market account modeled on the state space Ω={ω1,…,ωk}\Omega=\{\omega_{1},...,\omega_{k}\} is arbitrage free. Then this model is complete if and only if the k×(n+1)k\times(n+1) matrix AA given by

A=(1+rS11⁢(ω1)⋅⋅⋅S1n⁢(ω1)1+rS11⁢(ω2)⋅⋅⋅S1n⁢(ω2)⋅⋅⋅⋅⋅⋅⋅⋅⋅1+rS11⁢(ωk)⋅⋅⋅S1n⁢(ωk))A=\left(\begin{array}[]{cccccc}1+r&S_{1}^{1}(\omega_{1})&\cdot&\cdot&\cdot&S_{% 1}^{n}(\omega_{1})\\ 1+r&S_{1}^{1}(\omega_{2})&\cdot&\cdot&\cdot&S_{1}^{n}(\omega_{2})\\ \cdot&\cdot&&&&\cdot\\ \cdot&\cdot&&&&\cdot\\ \cdot&\cdot&&&&\cdot\\ 1+r&S_{1}^{1}(\omega_{k})&\cdot&\cdot&\cdot&S_{1}^{n}(\omega_{k})\end{array}\right)

has the rank kk, i.e. r⁢a⁢n⁢k⁢(A)=krank(A)=k.

Proof.

The model is complete if one can find a replicating strategy (x,ϕ)(x,\phi) for any claim XX. In other words, the model is complete if for any X∈ℝkX\in\mathbb{R}^{k} there exists a solution to the system of linear equations:

(1+rS11⁢(ω1)⋅⋅⋅S1n⁢(ω1)1+rS11⁢(ω2)⋅⋅⋅S1n⁢(ω2)⋅⋅⋅⋅⋅⋅⋅⋅⋅1+rS11⁢(ωk)⋅⋅⋅S1n⁢(ωk))⁢(x−∑i=1nϕi⁢S0iϕ1⋅⋅⋅ϕn)=(V1⁢(x,ϕ)⁢(ω1)V1⁢(x,ϕ)⁢(ω2)⋅⋅⋅V1⁢(x,ϕ)⁢(ωk)).\left(\begin{array}[]{cccccc}1+r&S_{1}^{1}(\omega_{1})&\cdot&\cdot&\cdot&S_{1}% ^{n}(\omega_{1})\\ 1+r&S_{1}^{1}(\omega_{2})&\cdot&\cdot&\cdot&S_{1}^{n}(\omega_{2})\\ \cdot&\cdot&&&&\cdot\\ \cdot&\cdot&&&&\cdot\\ \cdot&\cdot&&&&\cdot\\ 1+r&S_{1}^{1}(\omega_{k})&\cdot&\cdot&\cdot&S_{1}^{n}(\omega_{k})\end{array}% \right)\ \left(\begin{array}[]{c}x-\sum_{i=1}^{n}\phi^{i}S_{0}^{i}\\ \phi^{1}\\ \cdot\\ \cdot\\ \cdot\\ \phi^{n}\end{array}\right)=\left(\begin{array}[]{c}V_{1}(x,\phi)(\omega_{1})\\ V_{1}(x,\phi)(\omega_{2})\\ \cdot\\ \cdot\\ \cdot\\ V_{1}(x,\phi)(\omega_{k})\end{array}\right).

This shows that computation of a replicating strategy for the contingent claim XX is equivalent to solving the equation A⁢Z=XAZ=X, where ZZ is an n+1n+1 dimentional vector:

Z=(x−∑i=1nϕi⁢S0iϕ1⋅⋅⋅ϕn).Z=\left(\begin{array}[]{c}x-\sum_{i=1}^{n}\phi^{i}S_{0}^{i}\\ \phi^{1}\\ \cdot\\ \cdot\\ \cdot\\ \phi^{n}\end{array}\right).

By standard linear algebra the matrix AA has the rank kk if and only if for every X∈ℝkX\in\mathbb{R}^{k} the equation A⁢Z=XAZ=X has a solution Z∈ℝn+1Z\in\mathbb{R}^{n+1}. Hence the statement of the proposition follows immediately. ∎

Example 1.2.24.

We have already seen that the stochastic volatility model discussed in Examples 1.2.15 and 1.2.19 is not complete. Another way to see this is by using Proposition 1.2.23. The matrix AA in this case has the form

A=(1+r(1+h)⁢S01+r(1−h)⁢S01+r(1+l)⁢S01+r(1−l)⁢S0)A=\left(\begin{array}[]{cc}1+r&(1+h)S_{0}\\ 1+r&(1-h)S_{0}\\ 1+r&(1+l)S_{0}\\ 1+r&(1-l)S_{0}\end{array}\right)

The rank of this matrix is 22 which is not equal to k=4k=4. Hence it doesn’t have full rank, and the model is incomplete. ∎

Proposition 1.2.23 presents a method to determine whether a model is complete without computing replicating strategies. Now, if the model is incomplete, is there a method to determine whether a specific contingent claim is attainable without trying to compute the replicating strategy? Yes, there is. The following proposition shows how.

Proposition 1.2.25.

The contingent claim XX is attainable if and only if 𝔼ℚ⁢(11+r⁢X)\mathbb{E}_{\mathbb{Q}}\left(\frac{1}{1+r}X\right) takes the same value for all ℚ∈𝕄\mathbb{Q}\in\mathbb{M}.

The proof of this result can be found in Pliska ([1], page 23). We omit the proof here. An important consequence of this proposition is the following theorem:

Theorem 1.2.26.

Under the assumption that the model is arbitrage free, it is complete if and only if 𝕄\mathbb{M} consists of only one element, i.e. there is only one risk neutral measure.

Proof.

Since the model is arbitrage free, it follows from Theorem 1.2.8 that there is at least one risk neutral measure, i.e. 𝕄≠∅︀\mathbb{M}\neq\emptyset. Assume first that there is only one risk neutral measure. Then the condition in Proposition 1.2.25 is trivially satisfied for all contingent claims XX and so the market model is complete.

Assume now the market model is complete and there are two risk neutral measures ℚ1\mathbb{Q}_{1} and ℚ2\mathbb{Q}_{2} in 𝕄\mathbb{M}. For each i=1,..,ki=1,..,k consider the contingent claim XiX^{i} given by

Xi⁢(ω)={1+r, if ⁢ω=ωi0, otherwise X^{i}(\omega)=\left\{\begin{array}[]{cl}1+r,&\mbox{ if }\omega=\omega_{i}\\ 0,&\mbox{ otherwise }\end{array}\right.

Since the model is complete, XiX^{i} is an attainable contingent claim. It follows from Remark 1.2.18 that

𝔼ℚ1⁢(11+r⁢Xi)=𝔼ℚ2⁢(11+r⁢Xi).\mathbb{E}_{\mathbb{Q}_{1}}\left(\frac{1}{1+r}X^{i}\right)=\mathbb{E}_{\mathbb% {Q}_{2}}\left(\frac{1}{1+r}X^{i}\right).

But clearly,

ℚ1⁢(ωi)=𝔼ℚ1⁢(11+r⁢Xi),ℚ2⁢(ωi)=𝔼ℚ2⁢(11+r⁢Xi),\mathbb{Q}_{1}(\omega_{i})=\mathbb{E}_{\mathbb{Q}_{1}}\left(\frac{1}{1+r}X^{i}% \right),\qquad\mathbb{Q}_{2}(\omega_{i})=\mathbb{E}_{\mathbb{Q}_{2}}\left(% \frac{1}{1+r}X^{i}\right),

which together implies that ℚ1⁢(ωi)=ℚ2⁢(ωi)\mathbb{Q}_{1}(\omega_{i})=\mathbb{Q}_{2}(\omega_{i}). Therefore ℚ1=ℚ2\mathbb{Q}_{1}=\mathbb{Q}_{2} and we have hence shown, that 𝕄\mathbb{M} consists of only one risk neutral measure. ∎

2. Multi-Period Market Models

2.1. General Model Specifications

The two most important new features of multi period market models as compared to single period market models are:

  • •
    ​

    Agents can buy and sell assets not only at the beginning of the trading period, but at any time tt out of a discrete set of trading times t∈{0,1,2,…,T}t\in\{0,1,2,...,T\}. t=0t=0 is the beginning of the trading period, t=Tt=T is the end.

  • •
    ​

    Agents can gather information over time, since they can observe prices. For example they can make their investments at time t=1t=1 dependent on the prices of the asset at time 11. These are unknown at time t=0t=0 and couldn’t be used in order to choose the investment at time t=0t=0.

These two aspects need special attention. We have to model trading as a dynamic process as opposed to the static trading approach in single period market models, and we have to take care about how the level of information evolves over time. The second aspect leads to the probabilistic concept of σ\sigma-algebras and filtrations. We still assume that we work on a finite state space Ω\Omega on which a probability measure ℙ\mathbb{P} is defined.

Definition 2.1.1.

A collection ℱ\mathcal{F} of subsets of the state space Ω\Omega is called a σ\sigma-algebra if the following conditions hold:

  1. (1)
    ​

    Ω∈ℱ\Omega\in\mathcal{F}

  2. (2)
    ​

    If F∈ℱF\in\mathcal{F} then Fc=Ω∖F∈ℱF^{c}=\Omega\setminus F\in\mathcal{F}

  3. (3)
    ​

    If Fi∈ℱF_{i}\in\mathcal{F} for i∈ℕi\in\mathbb{N}, then ⋃i=1∞Fi∈ℱ\bigcup_{i=1}^{\infty}F_{i}\in\mathcal{F}.

In our case, where the state space is finite, the third condition can be relaxed to condition

  • (3’)
    ​

    If F,G∈ℱF,G\in\mathcal{F}, then F∪G∈ℱF\cup G\in\mathcal{F}.

There are a few properties that can be easily derived from the above definition:

  • •
    ​

    ∅︀∈ℱ\emptyset\in\mathcal{F},

  • •
    ​

    If F,G∈ℱF,G\in\mathcal{F}, then F∩G∈ℱF\cap G\in\mathcal{F}.

To see the first one, notice that by 1. A=Ω∈ℱA=\Omega\in\mathcal{F}. By virtue of 2. one gets Ac∈ℱA^{c}\in\mathcal{F}, but the compliment of the full space is an empty set, so ∅︀=Ac∈ℱ\emptyset=A^{c}\in\mathcal{F}. Take now F,G∈ℱF,G\in\mathcal{F}. It can be easily checked that

F∩G=(Fc∪Gc)c.F\cap G=(F^{c}\cup G^{c})^{c}.

This reformulation will be very useful. By (2) we obtain Fc,Gc∈ℱF^{c},G^{c}\in\mathcal{F}. Further, their sum Fc∪GcF^{c}\cup G^{c} is an element of ℱ\mathcal{F} (see (3’)). Applying (2) once more, we have (Fc∪Gc)c∈ℱ(F^{c}\cup G^{c})^{c}\in\mathcal{F}, which completes the proof of the second statement.

The idea is that a σ\sigma-algebra models a certain level of information. The elements of a σ\sigma-algebra ℱ\mathcal{F} are subsets of the state space Ω\Omega. Given that ℱ\mathcal{F} is chosen to model the level of information of an agent or an observer of any kind, then it is assumed that this agent can distinguish between two sets FF and GG which belong to ℱ\mathcal{F}, but not between the actual elements of FF or the actual elements of GG. In a way one could say, that if the agent looks at the state space Ω\Omega his resolution is not high enough to recognise the actual states ω\omega but only to see the sets belonging to the σ\sigma-algebra ℱ\mathcal{F}. One can think of the states ω\omega as atoms, and as the sets contained in the σ\sigma-algebra as molecules, which are built from the atoms. The agent can only see the molecules, but not the atoms. The larger the σ\sigma-algebra, the higher the resolution is, the more information is in the σ\sigma-algebra.

A σ\sigma-algebra on a finite state space can be decomposed into a so called partition:

Definition 2.1.2.

A partition of a σ\sigma-algebra ℱ\mathcal{F} is a collection of sets {Ai:i∈I}\{A_{i}:i\in I\}, where II is some index set, such that:

  1. (1)
    ​

    Ai≠∅︀A_{i}\neq\emptyset for i∈Ii\in I.

  2. (2)
    ​

    Ai∈ℱA_{i}\in\mathcal{F} for i∈Ii\in I.

  3. (3)
    ​

    Every set F∈ℱF\in\mathcal{F} can be written as a union of some of the AiA_{i}’s, i.e. F=⋃j∈JAjF=\bigcup_{j\in J}A_{j} with J⊂IJ\subset I.

  4. (4)
    ​

    Sets AiA_{i} are pairwise disjoint, i.e. Ai∩Aj=∅︀A_{i}\cap A_{j}=\emptyset whenever i≠ji\neq j.

The set II can be thought of as a set of integers 1,2,…,N1,2,\ldots,N for some NN. Therefore, a partition is a collection of sets A1,A2,…,ANA_{1},A_{2},\ldots,A_{N} satisfying conditions (1)-(4) of Definition 2.1.2. It can be easily deduced that all sets AiA_{i} in a partition satisfy a certain minimality condition: if A∈ℱA\in\mathcal{F} and A⊂AjA\subset A_{j} for some jj, then A=AjA=A_{j}, since otherwise it could not be written as a union of some of the AiA_{i}’s (a contradiction with (3)). It is not hard to show that given a σ\sigma-algebra ℱ\mathcal{F}, a partition for this σ\sigma-algebra always exist and is in fact unique.

Example 2.1.3.
  1. (1)
    ​

    If Ω\Omega is a finite state space, then the power set 𝒫⁢(Ω)\mathcal{P}(\Omega) is a σ\sigma-algebra. This is the largest possible σ\sigma-algebra on Ω\Omega, but beware: If Ω\Omega is not finite, then the power set of Ω\Omega is not a σ\sigma-algebra! The partition of ℱ\mathcal{F} consists of sets {ω}\{\omega\} for all ω∈Ω\omega\in\Omega.

  2. (2)
    ​

    The trivial σ\sigma-algebra corresponding to a state space Ω\Omega is the σ\sigma-algebra ℱ={∅︀,Ω}\mathcal{F}=\{\emptyset,\Omega\}. This clearly satisfies the conditions in Definition 2.1.1. The trivial σ\sigma-algebra is the smallest σ\sigma-algebra on Ω\Omega. Here, the partition is composed from one set Ω\Omega.

  3. (3)
    ​

    On a state space Ω={ω1,ω2,ω3,ω4}\Omega=\{\omega_{1},\omega_{2},\omega_{3},\omega_{4}\} consisting of four elements, the following is a σ\sigma-algebra:

    ℱ={∅︀,{ω1,ω2},{ω3,ω4},Ω}.\mathcal{F}=\{\emptyset,\{\omega_{1},\omega_{2}\},\{\omega_{3},\omega_{4}\},% \Omega\}.

    This can be easily verified. A partition of this σ\sigma-algebra, is given by the two sets A1={ω1,ω2}A_{1}=\{\omega_{1},\omega_{2}\} and A2={ω3,ω4}A_{2}=\{\omega_{3},\omega_{4}\}.

∎

Let us now consider a random variable X:Ω→ℝX:\Omega\to\mathbb{R} (we can think of a contingent claim). It is ℱ\mathcal{F}-measurable if its values depend only on information in ℱ\mathcal{F}. This is a general mathematical definition:

Definition 2.1.4.

The random variable X:Ω→ℝX:\Omega\to\mathbb{R} is called ℱ\mathcal{F}-measurable, if for every closed interval [a,b]⊂ℝ[a,b]\subset\mathbb{R} the preimage under XX belongs to ℱ\mathcal{F}, i.e.

X−1⁢([a,b])∈ℱ.X^{-1}([a,b])\in\mathcal{F}.

It is rather difficult to imagine and check. However, we can provide a simpler reformulation in our case of finite state space Ω\Omega. It is based on the observation that the set of values of XX is finite.

Proposition 2.1.5.

Let X:Ω→ℝX:\Omega\to\mathbb{R} be a random variable and (Ai)i∈I(A_{i})_{i\in I} a partition of the σ\sigma-algebra ℱ\mathcal{F}. The random variable XX is ℱ\mathcal{F}-measurable, if and only if XX is constant on each set of the partition, i.e. there exist cj∈ℝc_{j}\in\mathbb{R} for all j∈Ij\in I such that

X⁢(ω)=cj⁢ for all ⁢ω∈Aj.X(\omega)=c_{j}\mbox{ for all }\omega\in A_{j}.
Proof.

Assume (Ai)i∈I(A_{i})_{i\in I} is a partition of the σ\sigma-algebra ℱ\mathcal{F} and that XX is ℱ\mathcal{F}-measurable. Let j∈Ij\in I be an index and ω∈Aj\omega\in A_{j} be arbitrary. Define cj:=X⁢(ω)c_{j}:=X(\omega). Since XX is ℱ\mathcal{F}-measurable, we have X−1⁢(cj)∈ℱX^{-1}(c_{j})\in\mathcal{F} and hence by remarks following the definition of a σ\sigma-algebra, we have

∅︀≠X−1⁢(cj)∩Aj∈ℱ.\emptyset\neq X^{-1}(c_{j})\cap A_{j}\in\mathcal{F}.

Since clearly X−1⁢(cj)∩Aj⊂AjX^{-1}(c_{j})\cap A_{j}\subset A_{j}, we have by the aforementioned minimality property of the sets contained in the partition that

X−1⁢(cj)∩Aj=Aj.X^{-1}(c_{j})\cap A_{j}=A_{j}.

But this means nothing else than X⁢(ω)=cjX(\omega)=c_{j} for all ω∈Aj\omega\in A_{j} and hence XX is constant on AjA_{j}. By varying jj we obtain such cjc_{j}’s for all j∈Ij\in I.

Now assume vice versa, that X:Ω→ℝX:\Omega\to\mathbb{R} is a function which is constant on all sets AjA_{j} belonging to the partition and that the cjc_{j} are given as in the statement of the proposition. Let [a,b][a,b] be a closed interval in ℝ\mathbb{R} and define

C:={cj|j∈I⁢ and ⁢cj∈[a,b]}.C:=\{c_{j}|j\in I\mbox{ and }c_{j}\in[a,b]\}.

Since ⋃i∈IAi=Ω\bigcup_{i\in I}A_{i}=\Omega no other elements then the cjc_{j} occur as values of XX. Therefore

X−1⁢([a,b])=X−1⁢(C)=⋃j|cj∈CX−1⁢(cj)=⋃j|cj∈CAj.X^{-1}([a,b])=X^{-1}(C)=\bigcup_{j|c_{j}\in C}X^{-1}(c_{j})=\bigcup_{j|c_{j}% \in C}A_{j}.

Finally, due to property 3. of a σ\sigma-algebra we have

⋃j|cj∈CAj∈ℱ.\bigcup_{j|c_{j}\in C}A_{j}\in\mathcal{F}.

∎

In the proposition above, it is not assumed that the cjc_{j}’s all have different values. In fact, some of them may coincide.

The main idea behind the notion of a filtration is to model the development of information in time. A filtration is a sequence of σ\sigma-algebras. We assume that information can only increase in time, i.e. we never forget anything!

Definition 2.1.7.

A sequence (ℱt)0≤t≤T(\mathcal{F}_{t})_{0\leq t\leq T} of σ\sigma-algebras on Ω\Omega is called a filtration, if ℱs⊂ℱt\mathcal{F}_{s}\subset\mathcal{F}_{t} whenever s<ts<t.

We interpret the σ\sigma-algebra ℱt\mathcal{F}_{t} as the information which is available to an agent or an observer at time tt. In particular ℱ0\mathcal{F}_{0} represents the information which is available at the beginning of the period before anything else has started. This is, in a way, the initial information. Usually we know everything about the present state of the market, i.e. the prices of assets. Therefore, we will be assume that ℱ0\mathcal{F}_{0} is a trivial σ\sigma-algebra ℱ0={∅︀,Ω}\mathcal{F}_{0}=\{\emptyset,\Omega\}.

To model a dynamic random behaviour of some quantities (e.g. asset prices), we introduce the concept of stochastic processes, which are sequences of random variables in time.

Definition 2.1.8.

A family (Xt)(X_{t}), where 0≤t≤T0\leq t\leq T, consisting of random variables is called a stochastic process. If (ℱt)0≤t≤T(\mathcal{F}_{t})_{0\leq t\leq T} is a filtration, the stochastic process (Xt)(X_{t}) is called (ℱt)(\mathcal{F}_{t})-adapted if for all tt we have that XtX_{t} is ℱt\mathcal{F}_{t}-measurable.

We shall often write “process” instead of stochastic process. If it is clear from the context which filtration is meant, we shall also write “adapted” instead of (ℱt)(\mathcal{F}_{t})-adapted. The stochastic process (Xt)(X_{t}) is therefore adapted, if and only if its time tt component XtX_{t} depends only on the information which is available at time tt, i.e. ℱt\mathcal{F}_{t}.

Example 2.1.9.

An ideal example of a stochastic process is the evolution of stock prices. We denote the stock price at times 0≤t≤T0\leq t\leq T with StS_{t}. Since these prices are not known at time 0, based on the initial information, StS_{t} are assumed to be random variables for t≥1t\geq 1:

St:Ω→ℝ+.S_{t}:\Omega\to\mathbb{R}_{+}.

At time tt however we know the price of StS_{t} (i.e. it is a part of the information available to us) and we therefore assume that StS_{t} is ℱt\mathcal{F}_{t}-measurable. We therefore model the stock price evolution as an adapted process (St)(S_{t}), where the filtration (ℱt)(\mathcal{F}_{t}) is otherwise specified. ∎

In the above example we discussed the requirements for the price process so that is complies with the information structure we have. It is usually assumed that the only information we get comes from the prices themselves. Below, we present a general method how to obtain a filtration, such that the process in question is measurable with respect to this filtration, and the filtration represents the information flow which is induced by observing the stochastic process.

Definition 2.1.10.

Let (Xt)0≤t≤T(X_{t})_{0\leq t\leq T} be a stochastic process on (Ω,ℙ)(\Omega,\mathbb{P}). Define

ℱtX:=σ(Xu−1([a,b]):0≤u≤t,a≤b∈ℝ)\boxed{\mathcal{F}_{t}^{X}:=\sigma\left(X_{u}^{-1}([a,b]):0\leq u\leq t,a\leq b% \in\mathbb{R}\right)} (2.1.1)

This is the smallest σ\sigma-algebra which contains all the sets Xu−1⁢([a,b])X_{u}^{-1}([a,b]) where 0≤u≤t0\leq u\leq t and a≤ba\leq b. Clearly ℱsX⊂ℱtX\mathcal{F}_{s}^{X}\subset\mathcal{F}_{t}^{X} and (ℱtX)0≤t≤T(\mathcal{F}_{t}^{X})_{0\leq t\leq T} is a filtration. It follows immediately from the definition that (Xt)0≤t≤T(X_{t})_{0\leq t\leq T} is (ℱtX)0≤t≤T(\mathcal{F}_{t}^{X})_{0\leq t\leq T}-adapted. (ℱtX)0≤t≤T(\mathcal{F}_{t}^{X})_{0\leq t\leq T} is called the filtration generated by the process XX.

Let us make the above definition more understandable. First, (2.1.1) reads as

ℱtX={the information generated by X0,X1,…,Xt}.\mathcal{F}^{X}_{t}=\{\text{the information generated by $X_{0},X_{1},\ldots,X% _{t}$}\}.

We know how to extract the information carried by one random variable (see Example 2.1.6). Here, we have t+1t+1 random variables. Let 𝒢i\mathcal{G}_{i} be the σ\sigma-algebra generated by XiX_{i}, i=0,1,…,ti=0,1,\ldots,t. Then ℱtX\mathcal{F}^{X}_{t} is the smallest σ\sigma-algebra containing 𝒢0,𝒢1,…,𝒢t\mathcal{G}_{0},\mathcal{G}_{1},\ldots,\mathcal{G}_{t}. It means that we put all sets from collections 𝒢0,…,𝒢t\mathcal{G}_{0},\ldots,\mathcal{G}_{t} into one bag called 𝒢\mathcal{G} and look for a smallest collection of sets (Ai)i∈I(A_{i})_{i\in I}, for some set of indexes II, that satisfies

  1. (1)
    ​

    Ai≠∅︀A_{i}\neq\emptyset for i∈Ii\in I.

  2. (2)
    ​

    Every set G∈𝒢G\in\mathcal{G} can be written as a union of some of the AiA_{i}’s, i.e. G=⋃j∈JAjG=\bigcup_{j\in J}A_{j} with J⊂IJ\subset I.

  3. (3)
    ​

    Sets AiA_{i} are pairwise disjoint, i.e. Ai∩Aj=∅︀A_{i}\cap A_{j}=\emptyset whenever i≠ji\neq j.

Actually, we can put into 𝒢\mathcal{G} solely the partitions of 𝒢0,…,𝒢t\mathcal{G}_{0},\ldots,\mathcal{G}_{t}. Notice that above conditions resemble those from Definition 2.1.2. Indeed, it is true that ℱtX\mathcal{F}^{X}_{t} is the σ\sigma-algebra whose partition is given by (Ai)i∈I(A_{i})_{i\in I}.
