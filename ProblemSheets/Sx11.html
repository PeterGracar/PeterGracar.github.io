Problem 2.1.

The correlation ρ\rho between assets AA and BB is 0.10.1. The other data is given in the following table:

Asset R¯\bar{R} σ\sigma
AA 10%10\% 15%15\%
BB 18%18\% 30%30\%
  1. (a)
    ​

    Find the proportions α\alpha of AA and (1−α)(1-\alpha) of BB that define a portfolio of AA and BB having minimum standard deviation.

  2. (b)
    ​

    What is the value of this minimum standard deviation?

  3. (c)
    ​

    What is the expected return of this portfolio?

Solution
  1. (a)
    ​

    We first calculate the covariance as

    σA⁢B=ρ⁢σA⁢σB=0.0045.\sigma_{AB}=\rho\sigma_{A}\sigma_{B}=0.0045.

    We are therefore looking for the minimiser of

    ℒ⁢(α)=12⁢(α2⁢σA2+(1−α)2⁢σB2+2⁢α⁢(1−α)⁢σA⁢B).\mathcal{L}(\alpha)=\frac{1}{2}(\alpha^{2}\sigma_{A}^{2}+(1-\alpha)^{2}\sigma_% {B}^{2}+2\alpha(1-\alpha)\sigma_{AB}).

    Note: We could have set the proportions of the two assets independently and added the condition that their sum needs to be equal to 1. This would lead to the same solution once we substitute the condition into the remaining equations.

    Taking the partial derivative and setting it to zero we have

    ∂ℒ∂α=α⁢σA2−(1−α)⁢σB2+σA⁢B−2⁢α⁢σA⁢B=0,\displaystyle\frac{\partial\mathcal{L}}{\partial\alpha}=\alpha\sigma_{A}^{2}-(% 1-\alpha)\sigma_{B}^{2}+\sigma_{AB}-2\alpha\sigma_{AB}=0,

    which is equivalent to

    α=σB2−σA⁢BσA2+σB2−2⁢σA⁢B=0.826.\alpha=\frac{\sigma_{B}^{2}-\sigma_{AB}}{\sigma_{A}^{2}+\sigma_{B}^{2}-2\sigma% _{AB}}=0.826.
  2. (b)
    ​

    The standard deviation is equal to

    S⁢D=α2⁢σA2+(1−α)2⁢σB2+2⁢α⁢(1−α)⁢σA⁢B⁢=α=0.826⁢0.139.SD=\sqrt{\alpha^{2}\sigma_{A}^{2}+(1-\alpha)^{2}\sigma_{B}^{2}+2\alpha(1-% \alpha)\sigma_{AB}}\overset{\alpha=0.826}{=}0.139.
  3. (c)
    ​

    The expected return is

    R¯=α⁢R¯A+(1−α)⁢R¯B≃0.114.\bar{R}=\alpha\bar{R}^{A}+(1-\alpha)\bar{R}^{B}\simeq 0.114.

Problem 2.3.

Suppose there are nn assets which are uncorrelated. You may invest in any one, or in any combination of them. The mean rate of return R¯\bar{R} is the same for each asset, but the variances are different. The return on asset ii has a variance σi2\sigma_{i}^{2} for i=1,…,ni=1,\dots,n.

  1. (a)
    ​

    Show the situation on a mean-variance diagram. Describe the efficient set.

  2. (b)
    ​

    Find the minimum-variance point. Express your result in terms of

    σ¯2=(∑i=1n1σi2)−1.\bar{\sigma}^{2}=\left(\sum_{i=1}^{n}\frac{1}{\sigma_{i}^{2}}\right)^{-1}.
Solution
  1. (a)
    ​

    The minimum-variance diagram is depicted in Figure 4. The efficient set is the red line in the diagram.

  2. (b)
    ​

    Since the assets are uncorrelated, we are solving the problem

    min\displaystyle\min ∑i=1nwi2⁢σi2\displaystyle\sum_{i=1}^{n}w_{i}^{2}\sigma_{i}^{2}
    s.t. ∑i=1nwi=1.\displaystyle\sum_{i=1}^{n}w_{i}=1.

    The corresponding Lagrangian is

    ℒ⁢(wi,λ)=12⁢(∑i=1nwi2⁢σi2)−λ⁢(∑i=1nwi−1).\mathcal{L}(w_{i},\lambda)=\frac{1}{2}(\sum_{i=1}^{n}w_{i}^{2}\sigma_{i}^{2})-% \lambda(\sum_{i=1}^{n}w_{i}-1).

    Takin partial derivatives, we obtain the system

    ∂ℒ∂wi=wi⁢σi2−λ=0\displaystyle\frac{\partial\mathcal{L}}{\partial w_{i}}=w_{i}\sigma_{i}^{2}-% \lambda=0 ⇔wi=λσi2\displaystyle\Leftrightarrow w_{i}=\frac{\lambda}{\sigma_{i}^{2}} i=1⁢…⁢n\displaystyle i=1\dots n
    ∂ℒ∂λ=0\displaystyle\frac{\partial\mathcal{L}}{\partial\lambda}=0 ⇔∑i=1nwi=1.\displaystyle\Leftrightarrow\sum_{i=1}^{n}w_{i}=1.

    Substituting the first nn equations into the last one, we have that it is equivalent to

    ∑i=1nλσi2=1⇔λ=(∑i=1n1σi2)−1=:σ¯2.\sum_{i=1}^{n}\frac{\lambda}{\sigma_{i}^{2}}=1\Leftrightarrow\lambda=\left(% \sum_{i=1}^{n}\frac{1}{\sigma_{i}^{2}}\right)^{-1}=:\bar{\sigma}^{2}.

    Therefore, the proportions of asset ii in the minimum variance portfolio is equal to wi=σ¯2σi2w_{i}=\frac{\bar{\sigma}^{2}}{\sigma_{i}^{2}}.
