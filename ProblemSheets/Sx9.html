Problem 9.1.

Consider the following single period market model with state space Ω={ω1,ω2,ω3}\Omega=\{\omega_{1},\omega_{2},\omega_{3}\}. There are two stocks available. The initial stock prices are S01=10S_{0}^{1}=10 and S02=6S_{0}^{2}=6. The stock prices evolve according to the following table:

ω1\omega_{1} ω2\omega_{2} ω3\omega_{3}
S11S_{1}^{1} 55 1010 1414
S12S_{1}^{2} 1010 1212 44

The probabilities of the states under the subjective measure ℙ\mathbb{P} are assumed to be ℙ⁢(ω1)=35\mathbb{P}(\omega_{1})=\frac{3}{5} and ℙ⁢(ω2)=ℙ⁢(ω3)=15\mathbb{P}(\omega_{2})=\mathbb{P}(\omega_{3})=\frac{1}{5}.

    ​

    Compute the expectation, the variance and the covariance matrix of the returns R1R^{1} and R2R^{2} of the stocks under the subjective probability measure. Hint: you need to compute the returns of the stocks first using the prices given.

  2. (b)
    ​

    We are solving the problem

    minw1,w2⁡Var⁡(R)\displaystyle\min_{w_{1},w_{2}}\operatorname{Var}(R) =w12⁢σ12+w22⁢σ22+2⁢w1⁢w2⁢σ12,\displaystyle=w_{1}^{2}\sigma_{1}^{2}+w_{2}^{2}\sigma_{2}^{2}+2w_{1}w_{2}% \sigma_{12},

Problem 9.2.

The state space consists of 33 states: Ω={ω1,ω2,ω3}\Omega=\{\omega_{1},\omega_{2},\omega_{3}\}. Prices of risky assets at t=0t=0 are S01=4S^{1}_{0}=4, S02=10S^{2}_{0}=10. Time t=1t=1 prices are given by the following table:

ω1\omega_{1} ω2\omega_{2} ω3\omega_{3}
S11S_{1}^{1} 33 55 77

  1. (a)
    ​

    Compute the returns R1R^{1}, R2R^{2} of the stocks.

  2. (b)
    ​

    Compute the expectation, the variance and the covariance matrix of the returns of the stocks.

  3. (c)
    ​

    Solve the Mean-Variance portfolio problem: find an optimal portfolio with the expected return equal to 50%50\%.

  4. (d)
    ​

    Using Lemma 3.4.2, find an optimal portfolio with the expected return equal to 100%100\%.

  5. (e)
    ​

    Find the Market Portfolio. What is the expected return of the market portfolio?

  6. (f)
    ​

    Find the price for the call option (S12−10)+(S^{2}_{1}-10)^{+} determined by the CAPM principle.

Solution
  1. (a)
    ​

    The returns are

    ω1\omega_{1} ω2\omega_{2} ω3\omega_{3}

  2. (b)
    ​

    We calculate like we did in Problem 9.1 and get

    𝔼⁢(R1)=512,𝔼⁢(R2)=512,𝔼⁢((R1)2)=0.3125,𝔼⁢((R2)2)=0.375,\mathbb{E}(R^{1})=\frac{5}{12},\quad\mathbb{E}(R^{2})=\frac{5}{12},\quad% \mathbb{E}((R^{1})^{2})=0.3125,\quad\mathbb{E}((R^{2})^{2})=0.375,

    which gives Var⁡(R1)=0.1389\operatorname{Var}(R^{1})=0.1389 and Var⁡(R2)=0.2014\operatorname{Var}(R^{2})=0.2014. The covariance is Cov⁡(R1,R2)=−0.1111\operatorname{Cov}(R^{1},R^{2})=-0.1111.

  3. (c)
    ​

    We are searching for the proportional weights w0,w1w_{0},w_{1} and w2w_{2} satisfying

    min\displaystyle\min Var⁡(R)\displaystyle\operatorname{Var}(R)
    s.t. 𝔼⁢(R)=ρ=0.5\displaystyle\mathbb{E}(R)=\rho=0.5
    w0+w1+w2=1.\displaystyle w_{0}+w_{1}+w_{2}=1.

    The Lagrangian is then

    ℒ⁢(w1,w2,λ)=12⁢(w12⁢σ12+w22⁢σ22+2⁢w1⁢w2⁢σ12)+λ⁢((1−w1−w2)⁢r+w1⁢R¯1+w2⁢R¯2−0.5).\mathcal{L}(w_{1},w_{2},\lambda)=\frac{1}{2}(w_{1}^{2}\sigma_{1}^{2}+w_{2}^{2}% \sigma_{2}^{2}+2w_{1}w_{2}\sigma_{12})+\lambda((1-w_{1}-w_{2})r+w_{1}\bar{R}^{% 1}+w_{2}\bar{R}^{2}-0.5).

    This leads to the equations

    ∂ℒ∂w1\displaystyle\frac{\partial\mathcal{L}}{\partial w_{1}} =w1⁢σ12+w2⁢σ12+λ⁢R¯1−λ⁢r=0\displaystyle=w_{1}\sigma_{1}^{2}+w_{2}\sigma_{12}+\lambda\bar{R}^{1}-\lambda r=0
    ∂ℒ∂w2\displaystyle\frac{\partial\mathcal{L}}{\partial w_{2}} =w2⁢σ22+w1⁢σ12+λ⁢R¯2−λ⁢r=0\displaystyle=w_{2}\sigma_{2}^{2}+w_{1}\sigma_{12}+\lambda\bar{R}^{2}-\lambda r=0
    ∂ℒ∂λ\displaystyle\frac{\partial\mathcal{L}}{\partial\lambda} =(1−w1−w2)⁢r+w1⁢R¯1+w2⁢R¯2−0.5=0\displaystyle=(1-w_{1}-w_{2})r+w_{1}\bar{R}^{1}+w_{2}\bar{R}^{2}-0.5=0

    Solving the system gives w1=0.702w_{1}=0.702 and w2=0.562w_{2}=0.562, which therefore also gives w0=1−w1−w2=−0.27w_{0}=1-w_{1}-w_{2}=-0.27.
