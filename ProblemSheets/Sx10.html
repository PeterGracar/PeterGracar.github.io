    a⁢(x)=−u′′⁢(x)u′⁢(x).a(x)=-\frac{u^{\prime\prime}(x)}{u^{\prime}(x)}.

    r⁢(x)=−x⁢u′′⁢(x)u′⁢(x).r(x)=-\frac{xu^{\prime\prime}(x)}{u^{\prime}(x)}.

  5. (e)
    ​

    The fundamental theorem of asset pricing states that a model is arbitrage-free if and only if there exists at least one risk-neutral measure.

    The set of risk-neutral measures is given by 𝒫+∩𝕎⟂\mathcal{P}^{+}\cap\mathbb{W}^{\perp}. The set of arbitrage strategies is given by 𝕎∩𝔸\mathbb{W}\cap\mathbb{A}. Therefore, the previous statement is equivalent to the following statement.

    The market model is arbitrage-free if and only if 𝒫+∩𝕎⟂\mathcal{P}^{+}\cap\mathbb{W}^{\perp} is non-empty. This statement is equivalent to 𝕎∩𝔸\mathbb{W}\cap\mathbb{A} being empty.
